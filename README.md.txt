# Online Retail II - Exploratory Data Analysis

## Objective

Perform Exploratory Data Analysis (EDA) on the Online Retail II dataset to understand customer purchasing behavior.

## Dataset

Online Retail II Dataset (UCI Machine Learning Repository)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed

- Basic dataset exploration
- Missing value analysis
- Duplicate analysis
- Top 10 products by quantity
- Top 10 products by revenue
- Sales by country
- Monthly revenue trend
- Correlation heatmap
- Outlier detection
- Business insights

## Files

- Online_Retail_II_EDA.ipynb
- EDA_Report.docx
- online_retail_II.xlsx
